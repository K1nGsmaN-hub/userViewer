Вопрос:
    Расскажите об основных функциях PHP получения данных из SQL запросов.

Ответ: 
    Самая первая функция это mysqli_connect. Она служит для соеденения с базой данных. В неё передается
    четыре параметра: host, login, password, DataBase name. Возращает булевое значение.

    Следующая функция это mysqli_query. Создает запрос к базе данных. Принимает два параметра: сохраненный в переменную,
    результат выполнения прошлой функции и SQL-запрос (можно передать переменную с запросом). Если SQL-запрос, это модификация 
    таблицы, то вернет булевое значение.

    Для обработки данных полученных из прошлой фенкции с SQL-запросом SELECT, нужен метод mysqli_fetch_array. Принимает один
    параметр: сохраненная в переменную, прошлая функция. Возрвращает ассоциативный массив строк.
